Issues I faced.

-One issue I ran into was my text size being too large on my reflection.html file. Another was my picture taking up far too much of the screen. I fixed the text sizing issue by matching the heading tags.
-Another issue I encountered was not being able to alter the styles in the body section of my external stylesheet. I found that placing it last in my code would work but whatever style was listed first in my code would not work. I believe it has something to do with the cascading order in which the programs applies the style but am unsure.

What editor was used?

What happened the first time you viewed your page in a browser to test it?

My first webpage loaded with text but did not display my image properly and it overwhelmed my text. It needed to be resized.

How did you finally manage to resize the image?

I had to edit my picture size down to thumbnail size (193x176px).
